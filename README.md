Proyecto sencillo que contiene formularios HTML estáticos y páginas relacionadas.

Uso
- Abrir `index.html` en un navegador web (doble clic o arrastrar al navegador).
- Si usas XAMPP: coloca la carpeta en `htdocs` (ya está en `c:\xampp\htdocs`) y visita `http://localhost/BC/P1inditexOnce/formulariosHTML/index.html`.

Estructura
- `index.html`: página principal con formularios.
- `aboutme.html`: página informativa.
- `README.md`: este archivo.
- `LICENSE`: licencia del proyecto (MIT).
